# Kitcherry Reservas — Roadmap

Aplicación web para organizar reservas, controlar mesas y conocer mejor a los clientes habituales de un negocio hostelero.

## Tecnologías

- PHP
- SQLite
- HTML
- CSS
- JavaScript

## Versiones

### 001-panel-sencillo
Estructura inicial del proyecto, base de datos, conexión SQLite y primeras pantallas del panel.

### 002-login
Login, logout, sesiones PHP y protección de páginas internas.

### 003-configuracion-negocio
Formulario para configurar el restaurante, horarios, capacidad y tono de comunicación.

### 004-mesas
Gestión de mesas, zonas y capacidades.

### 005-reservas
CRUD principal de reservas.

### 006-listado-profesional
Listado con filtros, buscador y estados por colores.

### 007-dashboard
Resumen diario de reservas, comensales, ocupación y avisos.

### 008-alertas-internas
Detección de problemas mediante reglas programadas.

### 009-vista-mesas
Vista visual sencilla de mesas por zonas.

### 010-clientes
Registro de clientes e historial de reservas.

### 011-fidelizacion
Clasificación básica de clientes habituales, inactivos o con riesgo.

### 012-correos
Confirmaciones y recordatorios por email.

### 013-ia-mensajes
IA solo para adaptar el tono de mensajes.

### 014-limpieza-inactivos
Archivo o revisión de clientes sin actividad.
